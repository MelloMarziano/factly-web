Política de Privacidad

Última actualización: 02 de Enero, 2026

1. Introducción

En Factly, desarrollado por RookSys, respetamos su privacidad y estamos comprometidos a proteger sus datos personales. Esta política de privacidad le informará sobre cómo cuidamos sus datos personales cuando visita nuestro sitio web o utiliza nuestra aplicación móvil y le informará sobre sus derechos de privacidad y cómo la ley lo protege.

2. Datos que Recopilamos

Podemos recopilar, usar, almacenar y transferir diferentes tipos de datos personales sobre usted, que hemos agrupado de la siguiente manera:

  • Datos de Identidad: incluye nombre, RNC/Cédula, nombre de usuario o identificador similar.
  • Datos de Contacto: incluye dirección de facturación, dirección de correo electrónico y números de teléfono.
  • Datos Financieros: incluye detalles de cuentas bancarias y tarjetas de pago (procesados de forma segura a través de proveedores certificados).
  • Datos de Transacciones: incluye detalles sobre pagos y facturas emitidas/recibidas.
  • Datos Técnicos: incluye dirección IP, datos de inicio de sesión, tipo y versión del navegador, configuración de zona horaria y ubicación.

3. Cómo Usamos sus Datos

Solo usaremos sus datos personales cuando la ley nos lo permita. Más comúnmente, usaremos sus datos personales en las siguientes circunstancias:

  • Para registrarlo como nuevo cliente.
  • Para procesar y entregar sus facturas electrónicas a la DGII.
  • Para gestionar nuestra relación con usted.
  • Para mejorar nuestro sitio web, productos/servicios, marketing y experiencias de los clientes.

4. Seguridad de los Datos

Hemos implementado medidas de seguridad apropiadas para evitar que sus datos personales se pierdan accidentalmente, se usen o se acceda a ellos de manera no autorizada, se alteren o se divulguen. Además, limitamos el acceso a sus datos personales a aquellos empleados, agentes, contratistas y otros terceros que tengan una necesidad comercial de conocerlos.

5. Retención de Datos

Solo conservaremos sus datos personales durante el tiempo que sea necesario para cumplir con los fines para los que los recopilamos, incluso para cumplir con cualquier requisito legal, contable o de informes ante la DGII (mínimo 10 años según el Código Tributario).

6. Sus Derechos Legales

Bajo ciertas circunstancias, tiene derechos bajo las leyes de protección de datos en relación con sus datos personales, incluyendo el derecho a solicitar acceso, corrección, borrado, restricción, transferencia, u objeción al procesamiento.
